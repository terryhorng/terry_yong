「泰瑞版小小輸入法 for Windows」使用說明

◎支援系統：32/64 bits Windows XP/Vista/7/8.x/10/Server

◎下載網址：
http://terryhung.pixnet.net/blog/post/26968247
重要提醒：
下載完成後，請務必在 ZIP 檔上按右鍵，選「內容」，然後點選「解除封鎖」按鈕（或勾選「解除封鎖」，再按「套用」），
然後再解壓縮，這樣才不會在執行「yong.exe」時出現警告，而且後續使用上比較不會遇到問題。

◎檔案來源：

1.程式核心：
使用大陸人士「周永」（dgod）先生撰寫的「小小輸入法」程式核心（2018.10.19 版本）。
主程式（yong.exe）的主檔名為作者名字─「永」的漢語拼音，圖案為「永」的小篆寫法。

2.外觀介面：
使用泰國網友「輕鬆使者」提供的 Skin，以貼近正體中文 Windows 內建的中文輸入法介面，更多 Skin 檔可至小小輸入法論壇下載。

3.對照表（碼表）：使用台灣網友「Terry」整理的對照表，各種對照表來源如下：
(1)倉頡：「中標倉頡」（中文標準交換碼全字庫 (CNS11643) 倉頡）、馬來西亞‧倉頡之友「第五代倉頡」（外加 Terry 改良的亂倉打鳥「符號對照表」）；
(2)注音：「中標注音」（中文標準交換碼全字庫 (CNS11643) 注音，2010.04.13 版本）；
(3)漢語拼音：以「彰化一整天」網友製作的「批次取代 Excel 檔」為工具，將 Terry 整理後的注音對照表轉為拼音對照表；
(4)行列30：行列輸入法的家；
(5)無蝦米：舊版蝦米族樂園網站上的「蝦米查碼」、「蝦米檔案」、「加字加詞檔」；
(6)大易四碼：使用瑲珩於自由軟體鑄造場發布的大易四碼；
(7)韓文：Ubuntu 正體中文站 討論區；
(8)英文：「小小輸入法」內建。

◎使用方式：
【開啟解說畫面】
▲解說畫面中顯示「泰瑞倉頡」的橫條即為「主視窗」（下述 4～8 必須開啟「主視窗」才會有作用）。

1.下載後，解壓縮（可解壓縮至 C:\ 或 D:\），再執行解壓縮後的「yong.exe」即可。
2.在工作列右邊的小蝸牛圖示上按滑鼠右鍵，選「結束」，即可關閉程式。
3.開啟／關閉輸入法主視窗：Ctrl + Alt + /。
4.中打／英打切換：左 Shift。
5.打繁出繁／打繁出简切換：Ctrl + Alt + S。
6.半型／全型切換：Shift + Space。
7.半型／全型標點符號切換：Ctrl + .（針對 Shift + `～Shift + = 的符號進行全半型切換）。
8.重複上一個字：Ctrl + Alt + G。
9.安裝為 Windows TSF 輸入法（安裝為內置版）方式：
細節請上泰瑞的部落格查看「★泰瑞版小小輸入法─安裝設置篇」一文。
如果要在 Win8/8.1/10 Modern UI APPs 底下使用，請務必安裝為內置版。

◎注意事項：

1.輸入法模式切換：
Ctrl + Shift + `：泰瑞倉頡；
Ctrl + Shift + 1：倉頡聯想（組字後，顯示相關字詞供使用者選擇；組字時，僅顯示 Big5 字元）；
Ctrl + Shift + 2：倉頡注音（組字後，顯示注音供使用者查看或選擇；惟僅限於 CJK 字元集之字元）；
Ctrl + Shift + 3：行列30；
Ctrl + Shift + 4：大易四碼；
Ctrl + Shift + 5：無蝦米；
Ctrl + Shift + 6：拼音（只適合用來反查拼音，不適合用來打字）；
Ctrl + Shift + 7：注音（只適合用來反查注音，不適合用來打字）；
Ctrl + Shift + 8：韓文（先打「.」再打英文拼音，可用來輸入日文平假名、片假名）；
Ctrl + Shift + 9：英文（用途為邊打字邊找相關字）。
Ctrl + \：以上模式逐項切換（免安裝版不建議使用此組熱鍵，否則遇到 Size 較大的碼表時，可能會導致程式崩潰）。
注意：
(1)必須先開啟輸入法主視窗才能進行切換。
(2)在 Word 中，Ctrl + 1、Ctrl + Alt + 1 已定義為其他作用，因此使用以上熱鍵組合已是最佳選擇。

2.開啟螢幕小鍵盤（軟鍵盤）：Ctrl + Alt + K
注意：
(1)開啟後必須以滑鼠點選符號，而且使用完畢後必須自行關閉視窗。
(2)在主視窗上的「鍵盤圖示」上按右鍵，可切換至不同類別之符號。

3.倉頡、行列30、無蝦米對照表支援字元：
(1)完整支援 Unicode CJK 字元集的漢字（20,902 個字）。
(2)完整支援 Unicode CJK Ext-A 字元集的漢字（6,582 個字）。
(3)完整支援 Unicode CJK Ext-B 字元集的漢字（42,711 個字）。
(4)完整支援 Unicode CJK Ext-C 字元集的漢字（4,149 個字）。
(5)完整支援 Unicode CJK Ext-D 字元集的漢字（222 個字）。
Windows Vista/7/8.x/10 內建的新細明體、新細明體-ExtB 可直接顯示 CJK Ext-A/B 字元集的漢字，
Windows XP 需加裝「新細明體更新套件」才能顯示 CJK Ext-A/B 字元集的漢字。
CJK Ext-C/D 字元集的顯示問題請上泰瑞的部落格查看「Windows 8 內建的新細明體已經支援 CJK Ext-C/D 字元集了！」一文。
行列30不需在 CJK Ext-B/C/D 字元編碼末尾加上〝i〞。
無蝦米不需在 CJK Ext-A/B/C/D 字元編碼末尾加上〝;〞。

4.注音、拼音對照表支援字元：
(1)完整支援 Unicode CJK 字元集的漢字（20,902 個字）。
(2)完整支援 Unicode CJK Ext-A 字元集的漢字（6,582 個字）。
(3)完整支援 Unicode CJK Ext-B 字元集的漢字（42,711 個字）。

5.大易四碼對照表支援字元：
(1)完整支援 Unicode CJK 字元集的漢字（20,902 個字）。
(2)完整支援 Unicode CJK Ext-A 字元集的漢字（6,582 個字）。

6.於 Win32、Win64 應用程式下皆可正常運作，但於某些遊戲、軟體下可能無法正常運作。
無法正常運作時，請先在小蝸牛圖示上按右鍵，選「輸出方式」→「剪貼簿」，再試試看！

7.重碼（有候選字）時，按空白鍵會直接輸出第 1 個候選字，如果要選第 2 個或之後的候選字，請記得打完拆碼後直接按相對應的數字，不要按空白鍵。
(1)大易使用者請用 '、[、]、-、\ 輸出第 2 個至第 6 個候選字；
（和 Windows XP 內建的大易一樣，候選字窗格由 0 開始排起，
所以，按空白鍵是出第 1 個候選字，按 1 是出第 2 個候選字…按 5 是出第 6 個候選字；
總之，直接看候選字窗格上出現的數字，再按相對應的數字就對了！
只是，因為大易將橫排數字鍵作為字根，所以選字時要用桌上型電腦鍵盤右邊的九宮格數字鍵，
如果是筆記型電腦，請用 ' [ ] - \ 選字）。
(2)無蝦米可用 1、2、3、4、5、6、7、8 輸出第 2 個至第 9 個候選字；
（和行易公司的輸入法產品一樣，候選字窗格由 0 開始排起，
所以，按空白鍵是出第 1 個候選字，按 1 是出第 2 個候選字…按 8 是出第 9 個候選字；
總之，直接看候選字窗格上出現的數字，再按相對應的數字就對了！）
(3)注音、漢語拼音可用空白鍵輸出（每一頁的）第 1 個候選字、用 Shift + 2～Shift + 0 輸出第 2 個至第 10 個候選字。

8.只接受以小寫英文字母進行組字。

9.於組字視窗或候選字視窗上，以上、下鍵移動到粉紅色字元上，再按 Alt + Enter 可連結至「萌典」查詢該字（需為中文單字）的解釋（泰瑞倉頡改用「中國哲學書電子化計劃」之字典）。

10.慣用者可於桌面上建立「yong.exe」的捷徑，以利快速開啟；
或在程式集的「啟動」裡建立「yong.exe」的捷徑，登入 Windows 後立即執行此輸入法。

◎特殊功能：

1.萬用碼：
`（Tab 上方的按鍵），不確定正確拆碼時，可用來進行模糊拆碼（一個「`」代表一個模糊碼）。

2.注音輸入：
除注音模式外，在其他中文模式下，可用「'」當作前導字元，然後模擬注音輸入。
注意：
(1)輸入完成時不要加空白鍵，不然會上第一個候選字；
(2)不要用英文鍵上方的數字鍵選字，請改用滑鼠、方向鍵（加空白鍵）或鍵盤右邊的數字鍵選字。

3.手動反查組字字根：
切換到想查的輸入法模式（倉頡、行列30、無蝦米、注音、拼音、大易皆可），
選取並複製想要查的字（必須是單一漢字，而且一定要先按 Ctrl + C 複製起來），
再直接按「Ctrl + /」，即可反查該字在對應模式下的所有拆碼；
接著，可用滑鼠、方向鍵（加空白鍵）或鍵盤右邊的數字鍵，將反查到的拆碼直接「上屏」。
注意：在注音模式以外的其他中文模式下，因為結合注音輸入的功能，
所以反查到的拆碼會包含注音拆碼，只是它們會以「@」開頭，建議無須理會它們。

4.自動反查組字字根：
細節請上泰瑞的部落格查看「泰瑞版小小輸入法─相關字詞篇」一文。

5.更多特殊功能請上泰瑞的部落格查看「小小輸入法特殊功能介紹」、「泰瑞版小小輸入法─常見問答篇」。
